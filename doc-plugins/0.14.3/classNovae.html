Read (or re-read) settings from the main config file.

void Novae::restoreDefaults ( void  )

Set up the plugin with default values.

This means clearing out the Pulsars section in the main config.ini (if one already exists), and populating it with default values. It also creates the default novae.json file from the resource embedded in the plugin lib/dll file.
