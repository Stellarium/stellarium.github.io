class  CompassMarks
 Main class of the Compass Marks plug-in.
 

Detailed Description

Stellarium helps the user get their bearings using the cardinal point feature - the North, South, East and West markers on the horizon. CompassMarks takes this idea and extends it to add markings every few degrees along the horizon, and includes compass bearing values in degrees.
