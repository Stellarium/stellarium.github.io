void MeteorShowers::restoreDefaults ( void  )

Set up the plugin with default values.

This means clearing out the MeteorShower section in the main config.ini (if one already exists), and populating it with default values. It also creates the default showers.json file from the resource embedded in the plugin lib/dll file.
