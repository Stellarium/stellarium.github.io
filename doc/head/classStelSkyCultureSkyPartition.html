Detailed Description

An optional skyculture element, configurable in the index.json.

Currently Stellarium supports two optional such systems per skyculture: Zodiac and LunarSystem. The Chinese define 28 Lunar Mansions of different sizes in equatorial coordinates, and use defining stars for the onset of a mansion. The sky is dissected into 28 slices which extend from pole to pole. What is still unclear is whether we should keep the link to the stars "live", or let the stars move out of epoch. Currently the lines are kept "live", following the idea that, with each new official star map "from now on the Mansions should be defined like this". //!
