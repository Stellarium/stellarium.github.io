00036 This is an implementation of the L1 theory, V1.1.
00037 My modifications are:
00038 1) do not calculate constant terms at runtime but beforehand
00039 2) unite terms with the same frequencies
00040 3) rearrange the terms so that calculation of the argument becomes easier
00041 4) substitute so that the independent variable becomes T=jd-2433282.5
00042 5) artificially blend out the usage of polynomials outside [-695+t0,695+t0]
00043    in order to avoid a discontinuity
