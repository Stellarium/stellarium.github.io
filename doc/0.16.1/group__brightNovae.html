Detailed Description

The Bright Novae plugin displays the positions some bright novae in the Milky Way galaxy.

Bright Novae Catalog

The novae catalog is stored on the disk in JSON format, in a file named "novae.json". A default copy is embedded in the plug-in at compile time. A working copy is kept in the user data directory.

Configuration

The plug-ins' configuration data is stored in Stellarium's main configuration file (section [Novae]).
