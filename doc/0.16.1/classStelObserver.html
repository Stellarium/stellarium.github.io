Vec3d StelObserver::getTopographicOffsetFromCenter ( void  ) const

Get the geocentric rectangular coordinates of the observer in AU, plus geocentric latitude φ'.

This is vector ρ from Meeus, Astron. Algorithms, 2nd edition 1998, ch.11, p.81f. The first component is ρ cos φ' [AU], the second component is ρ sin &phi&' [AU], the third is φ' [radians].
