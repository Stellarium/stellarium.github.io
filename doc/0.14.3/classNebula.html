  • Name
  • CatalogNumber
  • Magnitude
  • RaDec
  • AltAzi
  • Distance
  • Size
  • Extra (contains the Nebula type, which might be "Galaxy", "Cluster" or similar)
  • PlainText

    flagsa set of InfoStringGroup items to include in the return value.
