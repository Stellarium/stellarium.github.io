Detailed Description

This uses the list of (usually measured) horizon altitudes to define the horizon.

Define it with the following names in landscape.ini:

Parameters
landscape/ground_coloruse this color below horizon
landscape/polygonal_horizon_listfilename containing azimuths/altitudes, compatible with Carte du Ciel.
landscape/polygonal_angle_rotatezoffset for the polygonal measurement (different from landscape/angle_rotatez in photo panos, often photo and line are not aligned.)
