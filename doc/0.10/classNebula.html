Nebula support the following InfoStringGroup flags:

  • CatalogNumber
  • Magnitude
  • RaDec
  • AltAzi
  • Distance
  • Size
  • Extra1 (contains the Nebula type, which might be "Galaxy", "Cluster" or similar)
