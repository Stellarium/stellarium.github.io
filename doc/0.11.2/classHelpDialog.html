void HelpDialog::retranslate ( ) [virtual, slot]

Retranslate the content of the dialog.

Needs to be connected to StelApp::languageChanged(). At the very least, if the window is based on a Qt Designer file (.ui), the implementation needs to call the generated class' retranslateUi() method, like this: 

 if (dialog) 
    ui->retranslateUi(dialog);

Set a key and description.

Note:
group and description must be in English; this function takes care of translating them. Of course, they still have to be marked for translation using the N_() macro.
Parameters:
groupis the help group. e.g. "Movement" or "Time & Date"
oldKeyis the textual representation of the old key binding (in the case or re-mapping), e.g. "CTRL + H". Can be an empty string
newKeyis the textual representation of the key binding, e.g. "CTRL + H"
descriptionis a short description of what the key does
